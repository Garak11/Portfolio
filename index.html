Moon Animation

This website is an animation of a Moon shaped peice of the night sky decorated with stars and is probably my favourite which is why I'm writing a description for only this website. For the code, I used Math.random in script to create shooting stars randomly and @keyframes in CSS to create the animations. One of the most notable part of the process for me was when we used two overlapping, different coloured circles to create the cresent moon you can see in the final website - I think this was especially eye-catching because it was a new course of action to create a unique shape. One thing coding this taught me was the skill to use @keyframes, transform and other animation related code like animation-duration and opacity to generate the illusion of a shooting star falling down, becoming more transparent as it goes down, and ending with an opacity of 0%, which means it is invisible.
